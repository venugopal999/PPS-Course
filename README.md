## 📂 Repository Structure
- **Lectures/** → PDF lecture notes
- **LabPrograms/** → C source codes for lab sessions
- **Assignments/** → Practice problems and solutions
- **README.md** → Course overview and instructions

---

## 📝 Course Overview
The PPS course introduces fundamental programming concepts using the C language.  
Topics include:
- Basics of C programming
- Control structures (loops, conditionals)
- Functions and recursion
- Arrays, strings, and pointers
